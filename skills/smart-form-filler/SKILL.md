# 智能表单填充 Skill

**名称**: smart-form-filler
**描述**: 根据自然语言自动识别表单类型，提取字段，调用对应接口完成数据插入
**版本**: 1.0.0
**作者**: 你

## 功能

1. **表单类型识别**: 根据用户输入的自然语言，识别是客户信息、商品信息还是预约信息表单
2. **字段提取**: 从自然语言中提取对应字段的值
3. **接口调用**: 自动组装 JSON 请求体，调用对应的后端接口
4. **智能推断**: 对缺失字段进行合理推断或提示用户补充

## 支持的接口

### 1. 客户信息表单

- **接口**: `POST http://localhost:8080/api/customers`
- **字段**: `name` (客户姓名), `phone` (客户电话), `level` (客户等级)

### 2. 商品信息表单

- **接口**: `POST http://localhost:8080/api/products`
- **字段**: `productName` (商品名称), `price` (商品价格), `stock` (库存数量)

### 3. 预约信息表单

- **接口**: `POST http://localhost:8080/api/appointments`
- **字段**: `visitorName` (来访人姓名), `appointmentDate` (预约日期), `remark` (备注信息)

## 触发器

- 关键词: "新增", "添加", "创建", "录入", "客户", "商品", "预约", "form", "add", "create"
- 模式: 包含表单相关操作的自然语言描述

## 使用示例

用户输入: "新增一个客户，姓名张三，电话13800138000，等级VIP"
技能输出:

- 识别到的表单类型: 客户信息表单
- 目标接口完整地址: http://localhost:8080/api/customers
- 提取出的字段内容: name=张三, phone=13800138000, level=VIP
- 组装后的 JSON 请求体: {"name":"张三","phone":"13800138000","level":"VIP"}

## 配置

- `baseUrl`: 后端服务地址 (默认: http://localhost:8080)
- `timeout`: 请求超时时间 (默认: 5000ms)

## 实现原理

1. **自然语言解析**: 使用关键词匹配和正则表达式提取信息
2. **表单识别**: 基于关键词判断表单类型
3. **字段映射**: 将自然语言中的信息映射到对应字段
4. **接口调用**: 使用 HTTP POST 请求提交数据

## 安装

将此技能目录复制到 OpenClaw 的 skills 目录，然后启用它。
